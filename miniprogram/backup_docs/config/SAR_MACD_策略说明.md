# SAR+MACD默认策略配置说明

## 策略名称
**SAR+MACD双重确认趋势跟踪系统**

## 策略类型
**多空双向** - 根据市场情况自动选择做多或做空

## 入场条件

### 做多条件 (AND逻辑 - 必须同时满足)
1. ✅ 日线SAR < 收盘价 (做多日)
2. ✅ 15分钟SAR < 收盘价
3. ✅ 15分钟MACD **金叉** 或 **多头排列** (二选一)
   - 金叉: DIF[2] <= DEA[2] AND DIF[1] > DEA[1]
   - 多头: DIF > DEA

### 做空条件 (AND逻辑 - 必须同时满足)
1. ✅ 日线SAR > 收盘价 (做空日)
2. ✅ 15分钟SAR > 收盘价
3. ✅ 15分钟MACD **死叉** 或 **空头排列** (二选一)
   - 死叉: DIF[2] >= DEA[2] AND DIF[1] < DEA[1]
   - 空头: DIF < DEA

## 出场条件

### 多头出场 (OR逻辑 - 任一满足即可)
1. ✅ 15分钟SAR转向空头 (SAR从多头变为空头)
2. ✅ 15分钟MACD死叉

### 空头出场 (OR逻辑 - 任一满足即可)
1. ✅ 15分钟SAR转向多头 (SAR从空头变为多头)
2. ✅ 15分钟MACD金叉

## 资金管理

- **资金模式**: 固定金额 (50 USDT/单)
- **杠杆**: 3倍
- **止损**: 0.5% (硬止损,不可调整)
- **止盈**: 1.0%
- **最大持仓**: 3个

## 风险控制

- **每日最大亏损**: 15%
- **最大回撤**: 20%
- **连续亏损**: 5次后停止交易
- **冷却时间**: 60秒

## 时间周期

- **方向判断**: 日线 (1D)
- **入场确认**: 15分钟 (15m)
- **分析间隔**: 30秒

## MACD参数

- **快线周期**: 21
- **慢线周期**: 30
- **信号线周期**: 5

## SAR参数

- **加速因子步长**: 0.02
- **加速因子最大值**: 0.2

## 状态

- **默认启用**: ❌ (需要用户手动启用)
- **推荐程度**: ⭐⭐⭐⭐⭐
