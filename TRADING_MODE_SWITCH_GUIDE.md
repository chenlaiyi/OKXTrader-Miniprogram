# 前端交易模式切换指南

## 📱 在小程序中切换交易模式

### 方法1: 通过策略编辑器（推荐）

#### 步骤：
1. 打开小程序 → 点击"策略编辑"页面
2. 进入"基础设置"标签页
3. 向下滚动找到"🎯 交易模式"选项
4. 点击选择器，选择模式：
   - **AI辅助模式**: 🤖 AI智能分析
   - **纯策略模式**: ⚡ 直接基于技术指标
5. 点击右上角"保存配置"按钮

#### 界面说明：
- **选项1**: AI辅助模式（紫色渐变标签）
  - 描述：🤖 AI智能分析，考虑更多因素（适合策略验证）
  - 适合：初期测试、策略验证阶段

- **选项2**: 纯策略模式（橙色渐变标签）
  - 描述：⚡ 直接基于技术指标，快速响应，零AI成本（推荐生产环境）
  - 适合：生产环境、长期运行

## 🎯 模式对比总结

| 特性 | AI辅助模式 | 纯策略模式 |
|------|-----------|-----------|
| **响应速度** | 2-5秒 | 50-100ms |
| **AI成本** | 每次¥0.10 | ¥0 |
| **每小时成本** | ~¥12 | ¥0 |
| **每月成本** | ~¥8,640 | ¥0 |
| **依赖性** | 依赖AI API | 完全自主 |
| **推荐用途** | 策略验证 | 生产运行 |

## 📋 配置保存

保存策略配置时，`tradingMode` 会被保存到数据库：
```json
{
  "basic_config": {
    "tradingMode": "pure"  // "ai" 或 "pure"
  }
}
```

下次启动自动交易时，系统会使用保存的模式。

---

## 🚀 快速切换步骤

1. **进入策略编辑** → 基础设置 → 交易模式
2. **选择模式** → AI辅助 or 纯策略
3. **保存配置** → 点击右上角保存按钮

✅ 完成！交易模式已切换
